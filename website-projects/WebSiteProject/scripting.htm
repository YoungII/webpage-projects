Server-Side Scripting

A technology by which a server-side script is run on a web server to dynamically generate web pages. Examples of server-side scripting technologies include PHP, Ruby on Rails, Microsoft Active Server Pages, Adobe ColdFusion, Sun JavaServer Pages, and Microsoft .NET. Sever-side scripting differs from CGI in that it uses direct execution.

Direct Execution

The script is run either by the web server itself or by an extension module to the web server.
